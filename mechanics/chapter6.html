第6章: 振動運動

6.1 単振動

単振動は、物体が平衡位置を中心に往復運動する運動の一形態です。単振動の代表的な例として、ばね振り子や振り子運動が挙げられます。

単振動の基本的な式は以下の通りです：

\(x(t) = A \cos(\omega t + \phi)\)

ここで、\(x(t)\) は時間 \(t\) における変位、\(A\) は振幅、\(\omega\) は角周波数、\(\phi\) は初期位相です。

角周波数 \(\omega\) は次のように表されます：

\(\omega = \sqrt{\frac{k}{m}}\)

ここで、\(k\) はばね定数、\(m\) は物体の質量です。周期 \(T\) は \(\omega\) に関連し、\(T = \frac{2\pi}{\omega}\) となります。

6.2 減衰振動

減衰振動は、外部からの抵抗力が働くことで振幅が時間と共に減少する振動運動です。空気抵抗や摩擦などが減衰の原因となります。

減衰振動の運動方程式は以下のように表されます：

\(m \frac{d^2x}{dt^2} + c \frac{dx}{dt} + kx = 0\)

ここで、\(c\) は減衰係数です。この方程式の解は、以下のように減衰項が含まれる形になります：

\(x(t) = A e^{-\gamma t} \cos(\omega' t + \phi)\)

ここで、\(\gamma = \frac{c}{2m}\) は減衰係数、\(\omega' = \sqrt{\omega^2 - \gamma^2}\) は減衰後の角周波数です。

6.3 強制振動

強制振動は、外部から周期的な力を受けて行われる振動運動です。この外力が共振条件に達すると、共振現象が発生し、振幅が大きくなります。

強制振動の運動方程式は以下のように表されます：

\(m \frac{d^2x}{dt^2} + c \frac{dx}{dt} + kx = F_0 \cos(\omega_{\text{外}} t)\)

ここで、\(F_0\) は外力の振幅、\(\omega_{\text{外}}\) は外力の角周波数です。

共振が発生する条件は、外力の角周波数が自然振動数に等しい場合、つまり \(\omega_{\text{外}} = \omega\) のときです。このとき、振幅は極大となります。
